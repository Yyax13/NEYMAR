>Uma simples ferramenta de detectar Malwares

## Instação
`$ git clone https://github.com/Yyax13/NEYMAR `

`$ cd NEYMAR`

`$ pip -r reqs.txt`

`$ python main.py`

## Funcionalidades

Você pode usar NEYMAR para/como:

1. Detectar Malware
2. detectar virus
3. detectar arquivos maliciosos
